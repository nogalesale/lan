Bocaditos y Salados Horario: Martes y Jueves Mensualidad: 100 Bs/módulo Material: Se cancela en cada clase. El instituto entrega los ingredientes que se emplean para cada clase receta, se devuelve en cada clase “comprado” el producto elaborado. Se pide un cuaderno para las lecciones, fotocopias de las recetas (o las da la profesora), un sobre (para el cobro de cocina). El uniforme para cada clase es: mandil, gorro o pañoleta para la cabeza. Deben presentarse con el cabello recogido y las uñas cortas, limpias y sin esmalte.
